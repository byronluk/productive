# Productivity Tracker

### Description

A side project where I am building a productivity tracker. My goal is to allow users to create reoccuring habits and also create one-time todos.

_Front End_

* React, Redux, and Flow
* Bootstrap V4

_Back End_

* Express / Node
* Firebase

**npm start** to set up
